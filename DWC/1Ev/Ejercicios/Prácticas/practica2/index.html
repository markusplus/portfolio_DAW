Práctica 2: Callbacks y fechas

En esta práctica vamos a trabajar con callbacks y programación asíncrona para crear un reloj digital que muestre la hora actual. Implemenentaremos los siguientes pasos:

  • Implementar una función en la que cada segundo se actualice el reloj
  • Implementar un botón que al pulsarlo muestre la fecha actual y, al volverlo a pulsar, deje de mostrarlo

El botón tiene que tener el texto de "Ocultar fecha" cuando la fecha aparezca y "Mostrar fecha" cuando la fecha esté oculta.

Nota: Para la realización de algunos ejercicios, busca información acerca de:

  • getElementById().style.visibility
  • .addEventListener()
